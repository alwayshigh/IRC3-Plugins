# IRC3-Plugins

## Setup

Copy the contents of coinmarketcap config.ini into your bots own config.ini file. 

## Comands

!coin btc

!overview

!top10 [gainers|losers] [1h|24h|7d]

By default !top10 searches 24h gainers but can optionally search others.

!volume
